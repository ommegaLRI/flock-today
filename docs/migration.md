# Migration approach

Migration should target faithful semantic reconstruction, not pixel-perfect cloning.

Stitch preserves:

- offer,
- visible copy,
- section hierarchy,
- primary CTAs,
- trust signals,
- forms/analytics detection,
- brand feel.

Stitch normalizes:

- arbitrary div structure,
- section roles,
- spacing and layout recipes,
- brand tokens,
- safe edit policies.

The default output should be a static Vite React campaign site because it is portable and cheap to host.
